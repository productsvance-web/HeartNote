HeartNote · Vitals trend pages (mockup)
Five dedicated screens, one per numeric vital. Same chassis: today's value · chart at the user's selected filter · four stats · plain-English pattern read · source breakdown.

Behavior note (not visible in this static mockup): each chart anchors to today and shows the user's selected filter window — D (today's hours), W (last 7 days), M (last 30 days, today minus one), 6M (last 6 calendar months), Y (last 12). The chart can scroll backward in time when prior data exists, but tapping the filter pill always re-anchors to "today minus the selected window." X-axis labels never skip months on 6M / Y, never skip days on W.
